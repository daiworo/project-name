My Resource Guide

  • W3Schools CSS Tutorial: http://www.w3.org/Style/CSS is a CSS resource where you can find articles and tutorials in English about CSS. The articles explain about the history, common misunderstandings, how CSS is used, etc. Alot of the information on this website is old but the most recent one was from 2023, while the others were from years ago. The articles from 2023 might be the most accurate way of how CSS is used in this present-day. This website also has articles in other languages such as Arabic, Chinese, Danish, German, Greek, French, Hebrew, Italian, Spanish, Portuguese, and more.
  • MDN Web Docs CSS: https://developer.mozilla.org/en-US/docs/Web/CSS is another CSS resource that mainly explains about Cascading Style Sheets (CSS). People can also learn how to use CSS from this resource particularly if someone is a beginner. This article was just released on December 19, 2024.
  • CSS-Tricks: https://css-tricks.com/ has multiple articles if anybody wants to read articles related to CSS.
  • Code Academy: https://www.codecademy.com/resources/docs/css. If you click on the URL, it will take you to a webpage and it will explain the basic meaning of CSS. It will also explain the concepts of CSS.
  • Web.dev: https://web.dev/learn/css is a website solely for learning CSS. This website does not show what date it was created but it may have been updated recently to keep up with how CSS works presently.
